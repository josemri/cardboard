**Sace VR**

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)
[![Unity](https://img.shields.io/badge/Unity-2022.3+-blue.svg)](https://unity.com/)

A simple VR shooter made with Unity 6 and Google Cardboard XR Plugin for android

The project contains a small arcade-style VR game where the player looks around, aims with a center dot, and destroys incoming enemies using the Cardboard trigger input.
Check out the releases section for the APK or feel free to build from source.
